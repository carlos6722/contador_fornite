# Fortnite Hub: Contador & Tienda Diaria 🎮

![Fortnite Hub Preview](bg.png)

Una aplicación web moderna estilo "Glassmorphism" construida con Vanilla HTML, CSS y JavaScript para la comunidad de Fortnite. 

### ✨ Características
- **Cuenta Regresiva de Temporada:** Un reloj en vivo que marca el tiempo exacto hasta la próxima gran actualización o inicio de temporada de Fortnite.
- **Tienda de Objetos Diaria en Vivo:** Consumo de la [API pública de Fortnite](https://fortnite-api.com) para renderizar automáticamente todos los objetos, lotes, *jam tracks* y cosméticos disponibles en el juego en tiempo real.
- **Temporizador de Rotación (UTC):** Marca el tiempo restante exacto de la tienda antes del siguiente reinicio global diario.
- **100% Responsivo y Animado:** Interfaz adaptada tanto a ordenadores como a dispositivos móviles con estética limpia, efectos visuales tipo "glitch" y menús integrados.
- **Espacios para Monetización:** Plantilla estructurada con espacios dedicados asincrónicos para insertar publicidad (como Google AdSense) a los laterales o como banners inferiores.

### 🛠️ Tecnologías
- HTML5 y CSS3 Vainilla (Variables nativas, CSS Grid y Flexbox)
- JavaScript (Promesas, Fetch API, manipulación del DOM)

### 🚀 Instalación y Uso
1. Clona el repositorio a tu entorno local.
2. Abre `index.html` directamente en tu navegador (no necesita servidor ni dependencias externas).
3. *(Opcional)* Reemplaza las variables `ca-pub-XXXXXXXX` del archivo HTML por tu identificador de Adsense si quieres desplegar tu propio hospedaje con monetización.
